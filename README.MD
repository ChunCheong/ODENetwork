# ODENetwork

ODENetwork is a framework to simulate networks of ODEs.

## Getting Started

Try running main.py to have a taste of what is it doing.

### Prerequisites

1. NetworkX (https://networkx.github.io/)

2. JiTCODE (https://jitcode.readthedocs.io/en/v1.3.0/#)
